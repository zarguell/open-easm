# UI Parity Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring the React UI to parity with the backend EASM capabilities: asset profiles, risk, evidence, change history, certificate lifecycle management, pivot observability, and outbound source-of-truth feeds.

**Architecture:** Keep the UI as a quiet operator console, not a marketing surface. Add typed API clients first, then focused views that consume existing backend endpoints. Use `DESIGN.md` as the visual source of truth for colors, typography, spacing, radii, and component behavior; prefer readable panels and dense tables over dumping JSON, while keeping raw attributes available as a fallback.

**Tech Stack:** React 19, TypeScript, Vite, React Router, TanStack Query, ky, Tailwind CSS, lucide-react, existing FastAPI endpoints.

**Repository Constraint:** Do not commit or stage changes. The user explicitly allows file changes but does not want commits.

---

## Design Source Of Truth

Every UI task must start by reading `DESIGN.md` and matching its style guide. The existing app already follows much of it through `ui/src/DESIGN_TOKENS.ts`, Tailwind utility classes, and shared components. New screens should extend those patterns instead of inventing a parallel design language.

Required design constraints from `DESIGN.md`:

- Use the near-black canvas: `#101010`, `#1a1a1a`, `#222222`.
- Use electric teal `#00d992` as the primary accent, not a new accent palette.
- Use hairline borders for dense operational surfaces.
- Use entity colors consistently:
  - ASN `#f59e0b`
  - IP range `#f97316`
  - IP `#ef4444`
  - Hostname `#06b6d4`
  - Domain `#00d992`
  - Certificate `#a855f7`
  - Org `#94a3b8`
- Use compact operator typography: small headings, mono eyebrows, mono metric values.
- Keep radii modest: prefer `rounded-sm`/`rounded-md`; avoid large decorative cards.
- Tables and panels should be data-dense, scan-friendly, and readable at desktop and mobile sizes.
- New icon buttons should use lucide-react icons and visible tooltips or labels where needed.

Implementation rule: before adding new colors, spacing, or component styles, check whether `DESIGN.md`, `ui/src/DESIGN_TOKENS.ts`, or an existing shared component already provides the pattern.

## Backend Features That Need UI Parity

The backend now exposes or stores these capabilities:

- Asset inventory: `GET /api/assets/inventory`
- Asset change ledger: `GET /api/assets/changes`
- Source-of-truth outbound feed: `GET /api/assets/export.ndjson`
- Certificate inventory: `GET /api/certificates/inventory`
- Certificate summary: `GET /api/certificates/summary`
- Entity detail attributes include `asset_profile` and `certificate_profile`
- Pivot queue already exists, but needs clearer operational status around queued/completed/skipped/failed jobs
- Runs, targets, graph, config, alerts, and geo map already have UI coverage

Out of scope:

- Source-of-truth imports.
- Ownership/CMDB/cloud/IPAM import screens.
- Broad visual redesign.
- Adding a new test framework unless the executor explicitly confirms it is worth the extra dependency.

```ts
function cleanParams(params: Record<string, unknown>): Record<string, string> {
  const cleaned: Record<string, string> = {}
  for (const [key, value] of Object.entries(params)) {
    if (value === undefined || value === null || value === '') continue
    cleaned[key] = String(value)
  }
  return cleaned
}

export function useAssetInventory(params: AssetInventoryParams = {}) {
  return useQuery({
    queryKey: ['asset-inventory', params],
    queryFn: () =>
      api
        .get('assets/inventory', { searchParams: cleanParams({ limit: 100, offset: 0, ...params }) })
        .json<{ assets: AssetInventoryItem[] }>(),
  })
}

export function useAssetChanges(params: AssetChangesParams = {}) {
  return useQuery({
    queryKey: ['asset-changes', params],
    queryFn: () =>
      api
        .get('assets/changes', { searchParams: cleanParams({ limit: 100, offset: 0, ...params }) })
        .json<{ changes: AssetChangeEvent[] }>(),
  })
}
```

```tsx
interface AssetInventoryTableProps {

function levelClass(level: string | null): string {
  if (level === 'critical') return 'text-red-400'
  if (level === 'high') return 'text-orange-300'
  if (level === 'medium') return 'text-yellow-300'
  if (level === 'low') return 'text-blue-300'
  return 'text-mute'
}

export function AssetInventoryTable({ assets, selectedEntityId, onSelectAsset }: AssetInventoryTableProps) {
  if (assets.length === 0) {
    return <div className="text-sm text-mute py-8">No profiled assets found</div>

  return (
    <div className="overflow-auto rounded-md border border-hairline">
      <table className="w-full text-sm">

          <tr className="text-left text-xs uppercase tracking-wider text-mute">
            <th className="px-3 py-2">Asset</th>

            <th className="px-3 py-2">Risk</th>
            <th className="px-3 py-2">Sources</th>
            <th className="px-3 py-2">Evidence</th>
            <th className="px-3 py-2">Feed</th>
            <th className="px-3 py-2">Last Seen</th>
          </tr>
        </thead>
        <tbody>
          {assets.map((asset) => (
            <tr
              key={asset.entity_id}
              onClick={() => onSelectAsset(asset.entity_id)}
              className={`cursor-pointer border-b border-hairline hover:bg-canvas-soft ${
                selectedEntityId === asset.entity_id ? 'bg-canvas-soft' : ''
```

```tsx
interface AssetChangeTimelineProps {
```

```tsx
import { useMemo, useState } from 'react'
import { useAssetInventory } from '../../api/assets'
import { useDebounce } from '../../hooks/useDebounce'
import { SearchInput } from '../shared/SearchInput'
import { ErrorDisplay } from '../shared/ErrorDisplay'
import { Skeleton } from '../shared/Skeleton'
import { SlideOver } from '../shared/SlideOver'
import { AssetInventoryTable } from './AssetInventoryTable'
import { AssetDetailPanel } from './AssetDetailPanel'
import { AssetExportPanel } from './AssetExportPanel'

export function AssetInventoryView() {
  const [riskLevel, setRiskLevel] = useState('')
  const [confidenceLevel, setConfidenceLevel] = useState('')
  const [feedEligible, setFeedEligible] = useState('')
  const [searchQuery, setSearchQuery] = useState('')
  const debouncedSearch = useDebounce(searchQuery)
  const [selectedEntityId, setSelectedEntityId] = useState<string | null>(null)

  const { data, isLoading, isError, error, refetch } = useAssetInventory({
    risk_level: riskLevel || undefined,
    confidence_level: confidenceLevel || undefined,
    feed_eligible: feedEligible === '' ? undefined : feedEligible === 'true',
    limit: 500,
  })

  const assets = data?.assets ?? []
  const filteredAssets = useMemo(() => {
    if (!debouncedSearch) return assets
    const lower = debouncedSearch.toLowerCase()
    return assets.filter((asset) => asset.entity_value.toLowerCase().includes(lower))
  }, [assets, debouncedSearch])
  const selectedAsset = assets.find((asset) => asset.entity_id === selectedEntityId) ?? null

  return (
    <div className="flex h-full">
      <div className="flex-1 flex flex-col min-w-0 p-6 gap-4">
        <div className="flex items-center justify-between gap-3">
          <h1 className="text-lg font-semibold text-ink">Asset Inventory</h1>
          <AssetExportPanel />
        </div>

        <div className="flex flex-wrap gap-3">
          <SearchInput value={searchQuery} onChange={(e) => setSearchQuery(e.target.value)} placeholder="Search assets..." className="max-w-md" />
          <select value={confidenceLevel} onChange={(e) => setConfidenceLevel(e.target.value)} className="rounded-md border border-hairline bg-canvas px-3 py-2 text-sm text-ink">
            <option value="">All confidence</option>
            <option value="high">High confidence</option>
            <option value="medium">Medium confidence</option>
            <option value="low">Low confidence</option>
          </select>
          <select value={riskLevel} onChange={(e) => setRiskLevel(e.target.value)} className="rounded-md border border-hairline bg-canvas px-3 py-2 text-sm text-ink">
            <option value="">All risk</option>
            <option value="critical">Critical</option>
            <option value="high">High</option>
            <option value="medium">Medium</option>
            <option value="low">Low</option>
            <option value="none">None</option>
          </select>
          <select value={feedEligible} onChange={(e) => setFeedEligible(e.target.value)} className="rounded-md border border-hairline bg-canvas px-3 py-2 text-sm text-ink">
            <option value="">All feed states</option>
            <option value="true">Feed eligible</option>
            <option value="false">Held from feed</option>
          </select>
        </div>

        {isError && <ErrorDisplay message={error.message} onRetry={() => refetch()} />}
        {isLoading && <div className="space-y-3 py-4">{Array.from({ length: 8 }).map((_, i) => <Skeleton key={i} height="40px" />)}</div>}
        {!isLoading && !isError && (
          <AssetInventoryTable assets={filteredAssets} selectedEntityId={selectedEntityId} onSelectAsset={setSelectedEntityId} />
        )}
      </div>

      <SlideOver open={selectedAsset !== null} onClose={() => setSelectedEntityId(null)} title="Asset Detail">
        {selectedAsset && <AssetDetailPanel asset={selectedAsset} />}
      </SlideOver>
    </div>
  )
}
```

```ts
import { useQuery } from '@tanstack/react-query'
import api from './client'

export interface CertificateInventoryItem {
  entity_id: string
  fingerprint_sha256: string | null
  subject_cn: string | null
  issuer_organization: string | null
  not_before: string | null
  not_after: string | null
  validity_state: string | null
  deployment_state: string | null
  observed_endpoints: Array<Record<string, unknown>>
  risk: string | null
  reasons: string[]
  strength: string | null
}

export interface CertificateSummary {
  total: number
  by_risk: Record<string, number>
  by_deployment_state: Record<string, number>
  by_issuer_organization: Record<string, number>
}

export function useCertificateInventory(params: { target_id?: string; deployment_state?: string; risk?: string } = {}) {
  return useQuery({
    queryKey: ['certificate-inventory', params],
    queryFn: () => {
      const searchParams: Record<string, string> = {}
      if (params.target_id) searchParams.target_id = params.target_id
      if (params.deployment_state) searchParams.deployment_state = params.deployment_state
      if (params.risk) searchParams.risk = params.risk
      return api.get('certificates/inventory', { searchParams }).json<{ certificates: CertificateInventoryItem[] }>()
    },
  })
}

export function useCertificateSummary(targetId?: string) {
  return useQuery({
    queryKey: ['certificate-summary', targetId],
    queryFn: () => {
      const searchParams: Record<string, string> = {}
      if (targetId) searchParams.target_id = targetId
      return api.get('certificates/summary', { searchParams }).json<CertificateSummary>()
    },
  })
}
```

```tsx
function readRecord(value: unknown): Record<string, unknown> | null {
```
